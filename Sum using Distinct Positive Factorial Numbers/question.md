Given a positive integer n, return whether n can be written as the sum of distinct positive factorial numbers.

Example 1:
Input
n = 31
Output
true
Explanation
Since 31 = 4! + 3! + 1!

Example 2:
Input
n = 4
Output
false
Explanation
Since 4 = 2! + 2! but not distinct.

Example 3:
Input
n = 6
Output
true

Example 4:
Input
n = 29
Output
false
