Given a positive integer n, return whether n can be written as the sum of distinct positive factorial numbers.

Example 1: <br>
Input <br>
n = 31 <br>
Output <br>
true <br>
Explanation <br>
Since 31 = 4! + 3! + 1! <br><br>

Example 2: <br>
Input <br> 
n = 4 <br> 
Output <br> 
false <br> 
Explanation <br> 
Since 4 = 2! + 2! but not distinct. <br>  <br> 

Example 3: <br> 
Input <br> 
n = 6 <br> 
Output <br> 
true <br>  <br> 

Example 4: <br> 
Input <br> 
n = 29 <br> 
Output <br> 
false
